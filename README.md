# Business_Live
CRUD Operation
A business website, Users can see their upcoming events and attend live stream videos all over the world Moreover Payment for all events users can pay for all events and then only access the event through the mail id link for attending the Events for a specific date.
Language Handled --- Python- Django Framework, Html, Css, Javascript, Bootstrap4
Build with a responsive website and handled all resolution.
Live_Website_Bussiness
Explore to watch live business website

Steps to Run in Locally:

Step 1 Enable the existing  virtual env path for Command :

env\scripts\activate

python manage.py runserver

Deployed in Railway:
Need to run this command :

pip install gunicorn

pip freeze > requirements.txt

Step 3 Explore the Live website in your fingertip

https://bussinesssite-production.up.railway.app/
